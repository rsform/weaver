Alloy

Alloy is a language that fuses Rust's power, safety guarantees, and performance with Swift's ergonomics and TypeScript's type system innovations. It features unique systems like explicit effect tracking and an advanced trait system with defaults. It maintains portability and compatibility by transpiling to Rust code, meaning it runs pretty much anywhere Rust does, and can take advantage of the existing Rust ecosystem.

Example

effect IO {
	read_line() -> String
	print(String)
}

fn process(input: String) -> String {
	input
		|> trim()
		|> to_uppercase()
		|> add_exclamation()
}

fn main() -> Result[(), Error] with IO {
	print("Enter your name: ")
	let name = read_line()
	let processed = process(name)
	print("Hello, {processed}!")
	Ok(())
}
